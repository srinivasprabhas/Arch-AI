Install and configure shadcn/ui.

Add these shadcn components:
- Button
- Card
- Dialog
- Input
- Tabs
- Text Area
- Scroll Area

Do not modify the generated component/ui/* files after installation.

Also install lucid-react.

Create lib/utils.ts with a reusable cn() helper for merging Tailwind classes.

Ensure all components match the existing dark theme in the globals.css

- All components import without errors
-  cn() works properly
- No default light styling appears
